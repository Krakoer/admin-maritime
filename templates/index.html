Getting started

Cette interface permet de lancer automatiquement une simulation d'attaque sur le banc de test. Sur cette page, vous trouverez les étapes pour utiliser cette plateforme.

Configuration des agents

Pour commencer, les agents Caldera installés sur les éléments du banc de test doivent être configurés. Pour cela, rendez vous dans l'onglet "agents" où les agents disponibles seront listés. Choisissez le rôle de chaque agent, et enregistrer les modifications.

Lancement d'une simulation

Une fois les agents configurés, l'onglet "simulation" permet de configurer et de lance r une simulation d'attaque.
